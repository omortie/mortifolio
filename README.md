# Developer Portfolio

This project provides a minimal starting point for building a modern developer portfolio with Next.js and Tailwind CSS. The main page now includes example content such as a profile image, sample experience entries and project screenshots.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the page.

Feel free to replace the placeholder text and images with your own content to personalize the portfolio.
